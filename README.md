# 🌠 Welcome Aboard

CosmosVerse is not a traditional website.

It is an interactive digital universe where users don't just **read** about space—they **experience** it.

Travel through the Solar System.

Discover distant galaxies.

Experience cinematic animations inspired by modern space exploration.
